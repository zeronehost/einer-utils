# einer-utils

工具库

## 安装

### npm

```bash
npm install einer-utils
```

### yarn

```bash
yarn add einer-utils
```

## API

- `camelUpperToLower`
- `camelLowerToUpper`
- `camelToLine`
- `lineToCamel`

## 示例

```javascript
import { camelUpperToLower } from "einer-utils";

console.log(camelUpperToLower("CamelUpperToLower")); // -> camelUpperToLower
```
